Image reconstruction

Example of Image reconstruction through inpainting using the heat equation. Initially, 97% of the image are lost. Using the remaining pixels as pinning points for a heat flow, one can recover a great deal of the information contained in the image.
The Matlab code is available at https://github.com/frank-roesler/Files/blob/master/Inpainting.m
